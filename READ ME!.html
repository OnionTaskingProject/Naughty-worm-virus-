🪱 Naughty Worm / Minhoca Cagadora

📄 File Description

The Naughty Worm virus uses polymorphism techniques in cache folders; it does nothing more than open a cmd window showing phrases related to poop. Created for educational and entertainment purposes.

✨ About the Project

  • Polymorphic: randomly selects one version of the code each run.
  • Creates “poop” logs in the Downloads folder. 💩
  • Displays funny messages in PowerShell/Command Prompt. 🪱
  • 100% safe and reversible.

🛠️ How to Test

  1. Use a Windows VM or sandbox folders to run the script safely without affecting your real PC.
  2. For safe testing, set up test folders, for example: C:\Test\TEMP, C:\Test\CACHE, C:\Test\Startup.
  3. Run the script in PowerShell: .\minhoca.ps1
  4. Observe the logs and messages without worrying about any damage.

🧹 How to Remove

  1. Delete minhoca.ps1 from TEMP, CACHE, and Startup folders.
  2. Delete the MinhocaLogs folder in Downloads.
  3. Delete the MinhocaSafada folder in %APPDATA%.
